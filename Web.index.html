Web development

A web developer's job is to create websites.While their primary role is to insure the website is visually appealing and easy to navigate, many web developer's are responsible for the website's performance and capacity.

Types of web developer's

There are three main types of developers.Here is the detail:
1.Front-end developers
These developers work on the visual part of the websites,the pages vistors see and interact with.They design the physical layout of each page , integrate graphics and use HTML and JAVA SCRIPT to enhance the site.
2.Back-end developers
These developers create the website's structure ,write code and verify their works.Their responsibilities also may include managing access points for others who need to manage a website's content.
3.Full stack developers:
They do the work of both front-end and back-end developers.These developers have the knowledge to build a complete website and may work for the organizations that don't have the budget for a large website team.

Web developer's task and responsibilities

If you are a web developer, you can work for a company, or as a freelancer taking on projects for individual clients.Your task will vary depending on your work situation, but your responsibilties also include ;
*Designing user interface and navigation menus
*Writing and reviewing code for sites, typically HTML ,XLM or JAVA SCRIPT.
*Testing web applications
*Collabrating with designers, developers and stakeholders.

Web developer career path

Many web developers start their careers with a single focus,usually front-end or back-end developments.They may move on to be full-stack developers or explore careers in related fields, including project management,computer programming or graphic design.
